Class Search::Repos < Object

The repo search service. Searches GitHub for repos using Octokit.rb. Can be used by calling a single class method or by creating a new instance and then calling the call instance method.

Instances of the repo search service can be initialized with their arguments and all subsequent calls made with that instance will use the initializing arguments unless otherwise specified.

Search GitHub for matching repos.

Parameters

  • args - Arguments for the search. Only the :q option is required.

    • :q - The actual search query.

    • :page - The page offset. Defaults to 0.

    • :per_page - The number of repos per page. Defaults to 100.

    • :order - 'desc' to sort by stargazers descending or 'asc' to sort by stargazers ascending. Otherwise sort by the default: relevance.

    • :language - The programming language to filter by.

    All other qualifiers supported by the GitHub API are supported. For more info see: developer.github.com/v3/search/#search-repositories.

def self.call(args = {})
  query = args.delete(:q)

  return Kaminari.paginate_array([], total_count: 0) if query.blank?

  client = Octokit::Client.new(access_token: args.delete(:access_token))

  options = search_options(args)

  query = build_query(query, args)

  results = client.search_repos(query, options)

  repos = results[:items].map { |result| Repo.new(result) }

  Kaminari.paginate_array(repos, total_count: results[:total_count]).page(options[:page]).per(options[:per_page])
end
